English summary

MiroFish is a web-based multi-agent simulation and prediction tool. The basic workflow is simple: add seed material, describe a scenario, generate a graph, run a simulation, and review the report.

If you are evaluating it in English, focus first on the workflow and self-hosting requirements rather than only the homepage tagline.

Fast path

  1. Understand what MiroFish AI does.
  2. Run a small first scenario.
  3. Check GitHub before self-hosting.
  4. Use focused seed material.
